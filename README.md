LIBH6NCSU
=========

ABOUT
-----

Libh6ncsu is a free software collection of tools for using Qt/C++ on
the client side, as a complement of libqtssu
(https://gitlab.com/g76r/libqtssu).

Its name stands for library (lib) Hallowyn (h6n) and client-side
utils (csu).
